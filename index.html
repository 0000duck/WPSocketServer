WP SOCKET SERVER

These are the commands to access the WP Socket Server

auth

authenticates you with the service, without this, you cannot do much.
Inputs: emailAddress, password
Outputs:
msg:Authentication Success

list

shows your list of connections
Inputs: None
Outputs:
con:connection index;connected:description;server;port

monitor

monitor the connection of choice, automatically stop monitoring other connections
Inputs: connectionId
Outputs:
msg:Now monitoring 20

stopmonitoring

stop monitoring the connection
Inputs: connectionId
Outputs:
msg:Could not stop monitoring 20

create

create a connection
Inputs: server, port, serviceType
Outputs:
msg:Created 20

connect

connect a connection
Inputs: connectionId
Output:
msg:Connecting to 20

destroy

destroy a connection
Inputs: connectionId
Outputs:
msg:Destroyed 20

disconnect

disconnection a connection
Inputs: connectionId
Outputs:
msg:Disconnected from 20
